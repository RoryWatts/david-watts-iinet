a graphics editor for the Pros 

Author: DAVID WATTS

Date: 09-06-1998

 Paint Shop Pro 5
Graphics editor for Windows95/98
Download from: www.jasc.com
Size: 6.85MB
Shareware (30-day free trial).
Cost: about $150
An essential program in any Web author's toolbox is a graphics editor which can crop, resize, adjust and even distort images for use on HTML pages. One of the most popular for Windows users for more than two years has been Paint Shop Pro.
The easy-to-use piece of shareware is now up to version 5 - released last month - and it looks and feels better than ever. Sometimes new versions of software can be disappointing because they offer little new other than a redesigned interface.
This is not the case with Paint Shop Pro. Reviewers at cnet and the Consumate Winsock site have been moved to compare PSP with the Rolls-Royce of graphics editors, Adobe Photoshop.
PSP is said to have 90 per cent of Photoshop's features for 20 per cent of the price - a telling statement.
Standard image tools are built into PSP as well as features that won't be found in Photoshop and similar competitors, including quick batch conversion capabilities for converting graphic files from one image type to another (JPEGs to Bitmaps or BMPs for instance), advanced screen capturing with automatic drag and drop, a built-in browser for quickly viewing an entire directory, colour replacer and masking tools, interlaced graphics support (for GIF 87a and 89a images) and vector-based image capabilities.
PSP even provides built-in browsers that allow users to preview the results different filters or manipulators will have on an image.
As in earlier versions, PSP offers support for more than 40 file formats, provides comprehensive tools that make it simple to perform image adjustments (like cropping and retouching photos) and includes a wide variety of painting and drawing tools.
Reviewers at cnet claim that the biggest highlight of the new version is support for maskable layers. The Layers tool simplifies image composition and manipulation by allowing the user to adjust each part of an image separately, without affecting any of the others. The Masks feature determines which colours are transparent, allowing underlying layers to show through.
The unique Custom Brushes and Picture Tubes features help users to create complicated images ideal for backgrounds.
Other Web creation features are the new Palette tool which includes improved GIF transparency and a prebuilt, ?browser-safe? 216-colour palette; and Animation Shop, which lets you easily manufacture GIF animations.
PSP is a medium-sized download in these days of bloatware like Netscape Communicator and Microsoft Internet Explorer 4.0. On installation it seems a bit slow to start, but once open it operates much quicker than previous versions.
Toolbars are easily manipulated and well signposted so that you can find out about the many functions by hovering over the icons. The help facilities have improved enormously with particular attention paid to online support on the Web.
Speed is not impaired when performing what might seem a heavyweight job like browsing through all images in a directory. Unlike earlier versions, PSP now lets the directory of images stay on screen while individual files are called.
Cropping, adjustment (of brightness for example), distortion, colour changes and manipulation all seem easy tasks because of PSP's boost in speed.
All that is really needed is a steady hand and a creative mind.
