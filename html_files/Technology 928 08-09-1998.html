Good time for users to update Net tools 

Author: DAVID WATTS

Date: 08-09-1998

 IT HAS been a fruitful time for Internet users who like to keep their software up-to-date.
A quick glance at the TUCOWS (The Ultimate Collection of Winsock Software) and Black Stump program repositories reveals some updates that fall into the must-have category of Net tools.
?WinZip 7.0 beta 3: The latest version of the compression utility has an easier and more colourful user interface that adds to its reputation for reliability and user-friendliness.
? CuteFTP 2.5: The popular FTP (file transfer protocol) utility is a must for Web publishers. It has jazzed up its looks slightly and added new sites to its connection facility. CuteFTP is facing increasing competition from the likes of FTP Voyager which also has a new release available.
? Communicator 4.06: Netscape has a beta of its version 4.5 browser suite out and about for appraisal. But the 4.06 release is meant as the real upgrade for full-time, less buggy use.
? Eudora Lite 3.06: The free version of Qualcomm's popular quality e-mail tool gets a revamp.
? HotDog Pro Suite 5.1 and HotDog Express 1.52: Sausage software has had to work hard in the crowded HTML editor market. The 5.1 is aimed at novice Web authors.
? Arachnophilia 3.5 Build 3812: One of the HTML editors HotDog has to fight.
It has won five stars - the highest rating - from the critics at TUCOWS.
? WinRescue 95 and WinRescue 98: Both have won five stars from TUCOWS.
These packages are designed as quick, easy ways to back up critical Windows system files.
