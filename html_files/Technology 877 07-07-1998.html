Windows 98 sets loose flood of unwanted interest 

Author: DAVID WATTS

Date: 07-07-1998

 WINDOWS 98 is making an impact on the Net less than a fortnight after its public release, but it is not the sort that Microsoft would want.
Complaints about a wide variety of perceived glitches in the upgraded operating system are rolling in to various newsgroups, along with floods of whinges and queries to Web sites specifically set up to provide user help.
The cnet News.com site reported late last week that Dell Computer, Compaq Computer and others including IBM and Toshiba, were posting workarounds, fixes and warnings on their Web sites about upgrading to Windows 98.
Upgrade glitches were not specific to Dell and Compaq but affected all vendors in some way, the report said.
Dell is advising users that the new features in Windows 98 will require, for some models, a new set of drivers - the critical software that allows hardware devices to talk to Windows.
In some cases, a BIOS update might also be needed. BIOS is the software that configures a computer in the starting, or boot-up, sequence immediately after the PC is turned on.
The newsgroup, microsoft.public.win98.pre-release, was set up several months ago to deal with the big number of users worldwide who either had beta or pre-release versions of Windows 98. Posts to the group have doubled since the public release on June 25. Questions on installation problems and FAT32 conversion of disks have dominated since then.
A growing number of queries relate to the compatibility of other programs with Windows 98. In many cases, problems are solved by following recommendations from other users who respond to posted queries.
A similar philosophy is at work at the Windows 98 Annoyances site on the Web, set up in March as an offshoot of the popular Windows 95 Annoyances pages.
Annoyances relies heavily on users to post workarounds and fixes to common problems.
Topics covered include: reducing clutter, customising, performance, troubleshooting, networking and third-party annoyances. A special section even deals with funny stories about users' experiences.
One of the highlighted sections at Annoyances is a description of how to remove the integration of the Internet Explorer 4.0 browser and the operating system. This integration is a key sales point of 98, allowing users to view their PC's operations as if they were browsing the Web.
It is also a key point of the US Department of Justice suit against Microsoft - due to be heard in September - which alleges Microsoft is anti-competitive because it ties the browser with its OS.
Annoyances gives users the chance to download a small program called ie-off which stops the OS using the browser as its main interface. The operation can be reversed, and Annoyances admits many users probably bought 98 because of the browser bundling and may not want to lose it.
Other help sites include Microsoft's own which allows users to search a database but can be slow, the Communique magazine site and Windows Magazine.
