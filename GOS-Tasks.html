Load the data, deal with duplicates, merge¶

What we know about the data files

  • Universities Australia Data Final 2016.csv is completely unique, they were all collected 20/12/2016
  • Universities Australia Data Final 2017.csv is completely unique, they were all collected 27/9/2017
  • Universities Australia Data Final 2016 2017.csv is a subset of 2016_2018_GOS_UA_Data.csv
  • 2016_2018_GOS_1055_UWA_Data.csv contains no unique data (i.e. all the rows are captured elsewhere)

Therefore, the combination of these data files will make up our final dataset. We will merge these files with the tasks and duties files we have obtained separately.

Defining our cohort¶

We can define our cohort as the set of the following:

  • Those who have most recently graduated from a Masters (Coursework)
  • Those who have graduated in public health
  • Those who have documented tasks and duties

This set can be found simply using the following codes for the relevant AGS variables:

Tasks and Duties¶

Let's look closer at the tasks and duties that our cohort perform. One initial way to look at this, is to look at the frequency of ngrams that our cohort report. A ngram is an amount of n-words, so this is in effect just counting the amount of times words or phrases come up.

Before doing this, we'll need to preprocess the text. Which includes doing the following things:

  • Turning words into a list of tokens e.g. "Hello, I'm Rory" -> ['Hello', ',' , 'I'm', 'Rory]
  • Changing all characters into lower-case characters e.g. "Hello" -> "hello"
  • Stripping punctuation e.g. "Hello, I'm Rory" -> "Hello I'm Rory"
  • Removing stop-words and very short words e.g. "This is a cat" -> "cat"

def penn2morphy(penntag):
    """ Converts Penn Treebank tags to WordNet. """
    morphy_tag = {'NN':'n', 'JJ':'a',
                  'VB':'v', 'RB':'r'}
    try:
        return morphy_tag[penntag[:2]]
    except:
        return 'n'
    
def preprocessing(text):
    # tokenize into words
    tokens = [word for sent in nltk.sent_tokenize(text) for word in nltk.word_tokenize(sent)]

    # lower capitalization
    tokens = [word.lower() for word in tokens]

    # remove words less than three letters
    tokens = [word for word in tokens if len(word) >= 3]
    
    # remove stop words
    #tokens = [word for word in tokens if word not in new_stop_words]

    # lemmatize
    tokens = [wnl.lemmatize(word,pos=penn2morphy(tag)) for word, tag in pos_tag(tokens)]
    preprocessed_text = ' '.join(tokens)

    return preprocessed_text
